# Uttar Pradesh Electricity Consumption Dashboard

A Streamlit-based interactive dashboard for analyzing daily electricity consumption trends in Uttar Pradesh using real-world state electricity data from India.

## Features
- Daily consumption trend analysis
- Monthly average consumption view
- KPI cards for average, peak, and minimum usage
- Date range filters
- Interactive charts and insights

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
